🐍 Python Coursework — Avery Walker
A collection of Python projects and assignments completed during CSC1402 (Programming Fundamentals) at Concordia University Texas. Projects are organized by category and demonstrate progression from basic syntax through functions, loops, data structures, and game logic.

📁 Repository Structure
🎮 games/
Interactive command-line games built with Python — including two versions of Hangman showing iterative development, Battleship with a randomized grid, and a full deck of cards simulation.
Skills demonstrated: Functions, loops, recursion, randomization, ASCII art, input validation, modular design
🧮 calculators/
Utility programs that take user input and compute real-world results — pay checks, hiking distance, batting averages, and basic arithmetic.
Skills demonstrated: Arithmetic operations, exception handling, input validation, f-strings, menu-driven programs
📊 data-and-structures/
Programs focused on data handling — test score tracking, string processing, character searching, and a searchable employee records system using tuples.
Skills demonstrated: Lists, tuples, string methods, loops, formatted output, data modeling

📈 Skills Progression
These assignments follow the arc of an introductory CS course:

Weeks 1–2 — Basic I/O, arithmetic, variables (basic_arithmetic.py, hike_calculator.py)
Weeks 3–4 — Loops, conditionals, exception handling (test_scores.py, batting_average.py, pay_check_calculator.py)
Weeks 5–6 — Functions and modular design (functions_char_finder.py, deck_of_cards.py)
Weeks 7+ — Data structures, strings, game logic (tuples_employee_records.py, hangman_graphic.py, battleship.py)
